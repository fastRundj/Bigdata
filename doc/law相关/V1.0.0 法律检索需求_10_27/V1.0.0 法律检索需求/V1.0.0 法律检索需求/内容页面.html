    公诉机关深圳市罗湖区人民检察院。 被告人许某，女，××年××月××日出生，汉族，高中文化，户籍地广东省汕尾市。因本案，于2014年6月21日被刑事拘留，同年7月10日被逮捕。现羁押于深圳市罗湖区看守所。 辩护人林焕鹏，广东××××律师事务所律师。 被告人郑某，男，××年××月××日出生，汉族，中专文化，户籍地广东省汕头市。因本案，于2014年6月21日被刑事拘留，同年7月10日被逮捕。现羁押于深圳市罗湖区看守所。

许某某与郑某某非法吸收公众存款罪一审刑事判决书

基本案情


为证明上述事实，公诉机关提交下列证据：1.书证：接受刑事案件登记表、立案决定书；抓获经过、归案说明；被告人的身份证明；电脑打印资料；复函；工商注册登记、营业执照等资料；刑事判决书、刑事裁定书；鉴定聘请书、鉴定结论通知书；许某发展客户资料；郑某发展客户资料等；2.证人证言：证人蒋某、杜某、黄某甲、甘某甲、张某甲、张某乙、李某甲、张某丙、徐某甲、王某甲、甘某乙、谢某甲、何某甲、李某乙、马某、郝某、黄某乙、谢某乙、王某乙、江某、何某乙、王某丙、朱某、高某、钟某、冯某、黄某丙、邱某、张某丁、赵某、范某、邓某、徐某乙、李某丙、黄某丁、宋某、刘某甲、刘某乙、黄某戊、李某丁、李某戊、卢某、周某、刘某丙、陈某、姜某的证言；3.被告人的供述与辩解：被告人许某、郑某的供述与辩解；4.鉴定结论：资产评估报告书：深圳市鹏信资产评估土地房地产估价有限公司出具的报告书。 公诉机关认为，被告人许某、郑某的行为已构成非法吸收公众存款罪。鉴于二被告人是从犯，依法应当减轻处罚；被告人郑某如实供述自己的罪行，依法可以从轻处罚。建议本院对被告人许某判处一年以上二年以下有期徒刑，并处罚金；建议本院对被告人郑某判处六个月以上一年零六个月以下有期徒刑，并处罚金。诉请本院依法判处。 被告人许某、郑某在法庭上对指控的上述事实及上述证据无异议，承认控罪，对公诉机关的量刑建议无异议。 被告人许某的辩护人辩称：被告人许某如实供述自己的罪行，认罪态度较好，有悔罪表现，是初犯、偶犯，是从犯，且其犯罪情节轻微，主观恶性较小，对社会的危害性较小，请求法庭从轻处罚。辩护人当庭提交了《深圳市员工参加社会保险清单》、《广东省明诚孝道文化促进中心证明》、组织机构代码证、民办非企业单位登记证、悔过书、保证人承诺书各一份，许某义工照若干，分别证实以下内容：被告人许某在广东邦家公司深圳分公司没有为许某购买社保，许某在公司地位低，作用小；许某现全职做公益义工，有悔罪表现，个人品德优良，有社会责任感，有爱心；被告人许某有心悔过；如判处被告人许某缓刑，案外人愿为其作担保。 经审理查明，公诉机关指控被告人许某、郑某的上述事实客观、真实，上述证据来源合法，且经当庭质证，予以采信。辩护人当庭提交的证据内容客观、真实，来源合法，予以采信。

审理经过


深圳市罗湖区人民检察院以深罗检刑诉（2014）2208号起诉书指控被告人许某、郑某犯非法吸收公众存款罪，于2014年10月8日向本院提起公诉。本院依法适用简易程序，实行独任审判，公开开庭审理了本案。深圳市罗湖区人民检察院指派代理检察员詹集美出庭支持公诉，被告人许某、郑某及辩护人林焕鹏到庭参加诉讼。现已审理终结。

事实认定


本院认为，被告人许某、郑某作为公司的直接责任人员，参与非法变相吸收公众存款，扰乱金融秩序，数额巨大，其行为均已构成非法吸收公众存款罪。公诉机关对被告人许某、郑某犯非法吸收公众存款罪的指控，事实清楚，证据确实、充分，应予以支持。在共同犯罪中，被告人许某、郑某起次要作用，是从犯，应当减轻处罚。被告人许某、郑某归案后如实供述自己的罪行，是坦白，可以从轻处罚。辩护人的辩护意见与事实相符，予以采纳。公诉机关的量刑建议与被告人的犯罪事实及犯罪情节相符，予以采纳。

裁判结果


依照《中华人民共和国刑法》第一百七十六条、第二十五条第一款、第二十七条、第六十七条第三款之规定，判决如下：

一、被告人许某犯非法吸收公众存款罪，判处有期徒刑一年零六个月，并处罚金人民币50000元（刑期自判决执行之日起计算。判决执行以前先行羁押的，羁押一日折抵刑期一日。即自2014年6月21日起至2015年12月20日止；罚金自本判决生效之日起一个月内缴纳）。 二、被告人郑某犯非法吸收公众存款罪，判处有期徒刑六个月，并处罚金人民币20000元（刑期自判决执行之日起计算。判决执行以前先行羁押的，羁押一日折抵刑期一日。即自2014年6月21日起至2014年12月20日止；罚金自本判决生效之日起一个月内缴纳）。 如不服本判决，可于接到判决书的第二日起十日内，通过本院或者直接向广东省深圳市中级人民法院提起上诉。书面上诉的，应当提交上诉状正本一份、副本二份。

代理审判员刘玉霞

二〇一四年十月二十二日

书记员沈斯琪
